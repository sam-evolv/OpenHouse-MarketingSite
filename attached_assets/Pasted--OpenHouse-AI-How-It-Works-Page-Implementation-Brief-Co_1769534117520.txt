# OpenHouse AI - "How It Works" Page Implementation Brief

## Context Prompt for Replit

You are building a new "How It Works" page for OpenHouse AI, a premium property-technology platform. This page will be added to the existing marketing website and must match the exact design system and quality level of the current site.

**What OpenHouse AI does:** It's an AI-powered resident portal that replaces paper handover folders for property developers. Developers upload documents, AI processes them, residents activate via NFC/QR, and developers get analytics on what residents are asking.

**Target audience:** Property developers in Ireland (B2B). This page must convince developers that OpenHouse AI will save them time, money, and legal risk.

**Design requirements:** 
- Dark theme matching existing site (bg: #0b0c0f)
- Gold accent color: #D4AF37 (brand-500)
- Premium, polished feel - NOT generic SaaS
- Lucide React icons ONLY (no emoji, no Unicode symbols)
- Inter font family
- Generous whitespace
- Subtle animations with ease-premium: cubic-bezier(0.16, 1, 0.3, 1)

**Critical:** Review the existing OpenHouse AI website styling before building. Match it exactly.

## ANIMATIONS

Use CSS or Framer Motion. All animations use:
- Duration: 300ms for hovers, 500ms for scroll reveals
- Easing: cubic-bezier(0.16, 1, 0.3, 1)

| Element | Animation | Trigger |
|---------|-----------|---------|
| Hero content | Fade in + slide up 20px | Page load |
| Scroll indicator | Gentle bounce | Continuous |
| Section content | Fade in + slide up 30px | Scroll into view |
| Journey steps | Staggered fade in (100ms delay each) | Scroll into view |
| Tool cards | Border glow + subtle lift on hover | Hover |
| Phone mockups | Subtle float (optional) | Continuous |
| CTA buttons | Scale to 0.98 | Active/pressed |

**Reduced motion:** Wrap animations in `@media (prefers-reduced-motion: no-preference)` or use Tailwind's `motion-safe:` prefix.

## NOTES FOR DEVELOPER

1. **Match existing site exactly** - Review the current OpenHouse AI website before building. Typography, spacing, and interaction patterns must be consistent.

2. **No emoji** - Use only Lucide React icons. The X and Check icons replace any ❌ or ✓ symbols.

3. **Premium feel** - Generous whitespace, subtle animations, gold accents used sparingly. This is enterprise software, not a startup landing page.

4. **Image placeholders** - Add placeholder containers for screenshots. These will be replaced with actual product screenshots.

5. **Performance** - Lazy load images, optimise animations for 60fps, minimise layout shift.

6. **Accessibility** - Ensure colour contrast meets WCAG AA, all interactive elements are keyboard accessible, animations respect prefers-reduced-motion.
